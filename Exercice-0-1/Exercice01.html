Exercice 0.1

Enoncé

  • A partir de deux variables (int) qui peuvent changer n'importe quand, créer une condition qui permet d'afficher dans la console le plus petit des deux nombres.
  • Si les nombres sont identique, afficher : "C'est les mêmes !!"

Notions ou méthodes javaScript utilisées

Après avoir créer et attribuer des valeurs aux variables intA et intB, j'utilise une instruction if pour voir si intA est inférieur à intB.
Si ce n'est pas le cas, j'utilise une instruction else if vérifiant si les deux variables sont égales.
Enfin si aucune des deux conditions précédentes n'est vérifiée, la valeur intB qui est donc la plus petite et l'instruction else me permet d'afficher intB dans la console.
