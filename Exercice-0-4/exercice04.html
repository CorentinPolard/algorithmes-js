Exercice 0.4

Enoncé

A partir d'une variable age et d'une variable height : Faire une condition qui permet de vérifier si la personne a + de 18 ans et mesure + de 140, dans ce cas :

  • Afficher dans la console : "Je peux rentrer dans le manège"
  • Sinon afficher dans la console : "Je n'ai pas le droit de rentrer dans le manège"

Notions ou méthodes javaScript utilisées

J'ai utilisé une instruction if vérifiant deux conditions grâce à && puis une instruction else si au moins l'une des deux conditions n'est pas vérifiée.
