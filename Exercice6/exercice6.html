Exercice 6

Enoncé

        Avec le moins de lignes de code possible, proposez moi un algorithme qui, pour un nombre donné, affiche la table de multiplication liée. Par exemple : Si je demande 1, je veux voir :
        1x1 = 1
        1x2 = 2
        etc Et ce jusqu'à 12

Notions ou méthodes javaScript utilisées

Pour cet exercice, j'utilise une boucle for avec une variable que j'initialise à 1 et que j'augmente de 1 à chaques fois que le code s'éxecute et qui ne s'arrête pas avant que i soit = 13.
Puis à chaques fois, le code va calculer puis afficher le nombre donné multiplié par i.
